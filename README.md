# Speak Like a Brazilian - PWA Landing Page

## Project Overview
Uma Landing Page de alta conversão projetada para vender o eBook "Speak Like a Brazilian". O projeto utiliza uma abordagem "Mobile-First" e "App-like", integrando gamificação nativa para engajar o usuário antes da compra.

## Tech Stack
* **Core:** HTML5, CSS3 (Modern Features), Vanilla JavaScript (ES6+).
* **Architecture:** JAMstack ready (Static styling, logic in client-side).
* **PWA:** Service Workers, Manifest.json, Offline capabilities.
* **Storage:** LocalStorage para persistência de progresso do jogo, streak e tema.

## Key Features
1.  **Interactive Gamification:** Minigame de vocabulário integrado para provar o valor do método imediatamente.
2.  **App-Like Experience:** Transições suaves, feedback tátil (vibração) e design responsivo.
3.  **Streak Counter:** Contador de dias baseado na primeira visita para criar senso de compromisso.
4.  **Dark Mode Default:** Estética elegante com persistência de preferência do usuário.
5.  **PWA Installable:** Lógica personalizada para instalação na tela inicial.

## Setup
Basta hospedar os arquivos em qualquer servidor estático (Vercel, Netlify, GitHub Pages). Certifique-se de servir via HTTPS para que o Service Worker funcione corretamente.
